PROJECTS

PRTracker

Created to fill the need for a centralized, at-a-glance PR tracking board, to see in realtime which PRs are open across any number of repositories. Built for the Meetup company hackathon in July 2017.

The idea for this project was concieved with a teammate, but I built the app start to finish using Node.js and a Zeit server, integrating with Slack's API and GitHub's webhook notifications. After the hackathon, the app has been used used by my Meetup team for PR tracking.
Check out the repo here.

Question Quail

Like Apples to Apples but with freeform questions and answers, Question Quail is a multiplayer question game developed with a small group for Web Development at Tufts University in the fall of 2016.

I worked primarily on front end and integrating database requests, and worked some with the MongoDB database on the back end as well.
Play the game here, or check out our repo here.

ClimbMate

An Android weather app designed for outdoorspeople, with features to add custom places by latitude/longitude, view current weather and compare to historical trends and your ideal conditions, and at a glance decide the best place to go on a given day.

I conceived and developed this project in the spring of 2017 as my final project for the Mobile Software Development class at AIT Budapest. This app was written in Java in Android Studio.
Check out the repo and the presentation here.

EyeDraw

EyeDraw uses the open source WebGazer eye tracking program to create a drawing application that draws based on where you look on the screen.

I worked in a team of two to develop this small project for a Tufts OOP/GUI class; I worked full stack on this project
Check out our repo here.
